# i3-dotfiles
A collection of configuration files for i3 Window Manager. 


# Install
These are the following apps you need to include:
- Feh
- Compton
- Zsh
- Polybar(Credit to huylenguyen on Github for Polybar config inspiration)
- Cava
- i3lock 
- ttf-font-awesome
